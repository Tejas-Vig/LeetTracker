# LeetTracker 🚀

A full-stack DSA problem tracking system built with a custom Java backend and interactive frontend dashboard.

---

## 🔥 Features

* ✅ Add, edit, delete problems (CRUD)
* 🔍 Search & filter by difficulty, topic, status
* 📊 Analytics dashboard (difficulty distribution, weakest topic)
* 💾 Persistent storage using JSON file system
* ⚡ Real-time updates without page reload

---

## 🛠 Tech Stack

* **Backend:** Java (Custom HTTP Server)
* **Frontend:** HTML, CSS, JavaScript
* **Data Storage:** JSON
* **Version Control:** Git & GitHub

---
